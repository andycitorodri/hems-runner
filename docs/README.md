## ⚠️ Convención

Estos documentos son **vivos**: actualízalos cuando cambies algo importante en el proyecto. Especialmente:

- Si despliegas en otro hosting → actualiza URLs en `PROJECT_STATE.md`
- Si cambias el flow de despliegue → actualiza `DEPLOY_RUNBOOK.md`
- Si tomas decisiones gráficas en Fase 2 → actualiza `GRAPHICS_STRATEGY.md`

Documentación obsoleta es peor que no tener documentación.
